Overview

AWS Unified Standard is a Cloud FinOps project I worked on during my internship at NextEra Energy. This project was to improve the accuracy and consistency of AWS budgeting strandards across the organization. The goal was to create one unified budgeting standard that multiple teams could follow instead of using different methods.

What I Did

  • Interviewed and gathered requirements from 8 different FinOps teams to understand how each team forecasted and budgeted AWS cloud spend.
  • Built an Excel evaluation matrix to compare each teamâ€™s budgeting process, including strengths, weaknesses, and gaps that affected accuracy.
  • Identified patterns across teams and used the findings to propose a standard budgeting framework teams can follow for a more accurate company-wide budget.
  • Reviewed AWS Savings Plans to optimize investments and assessed tools that could improve Savings Plan optimization.

Tools and Methods

  • Microsoft Excel (matrix scoring, comparison, documentation)
  • Stakeholder interviews and requirements gathering
  • FinOps criteria-based evaluation
  • AWS cost concepts (budgeting, forecasting, Savings Plans)

Outcome / Impact

The project supported a clearer, standardized approach for budgeting AWS cloud spend across teams. This reduces differences between budgeting styles and supports a more consistent and accurate forecast.

Skills Learned

  • Cloud FinOps fundamentals and cost optimization thinking
  • Interviewing stakeholders and translating needs into requirements
  • Building comparison frameworks (strength/weakness analysis)
  • Process documentation and standardization
  • Communicating recommendations across teams
